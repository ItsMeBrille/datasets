Hvordan fungerer det?

All datasets are free to use for personal projects. Contact me if you want to use them for commercial purposes.

The recommended way to use the datasets for web is to add them using their GitHub pages links. That way you will get live changes in the dataset. The structure of the datasets will always stay the same.
